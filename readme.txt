Важные замечания

- в объект $client.user записываются данные о пользователе по умолчанию, они же используются в сценарии как данные пользователя
    - $client.user.name - его имя
    - $client.user.email - его email

- в остальные объекты $client записываются аналогичные данные, получаемые из других каналов
    - $client.ch.hofftime - данные о пользователе, переданные в чат-виджет с сайта корпоративного портала Hoff
